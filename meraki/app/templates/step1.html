Follow these instructions to configure your Meraki switch.

1. Download the template using the button below
2. Go to www.meraki.cisco.com and log in
3. Once logged in go to the top right corner and click on your username
4. Scroll down to the section titled "API access" and generate an API key
5. Copy and paste the API key into the template.xlsx into worksheet 1, cell A2
6. Save the file and press next
